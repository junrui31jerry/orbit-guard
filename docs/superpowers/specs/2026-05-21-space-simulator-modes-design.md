# OrbitGuard 三模式太空模擬器設計規格

日期：2026-05-21
狀態：已完成對話設計，等待使用者審閱

## 目標

OrbitGuard 從單一衛星風險監測原型，擴充成有主選單的三模式太空模擬器。第一版完整強化「地球外太空模擬器」，另外兩個模式建立可直接操控太空船的原型場景，作為後續太陽系介紹與黑洞世界玩法的基礎。

三個模式為：

- 地球外太空模擬器：主要完成模式，保留衛星監測核心並加入任務、事件、掃描、避碰動畫與更合理的太陽系宇宙背景。
- 太陽系模擬器：玩家可控制太空船在太陽系預覽場景中飛行，靠近星體時看到簡短介紹。
- 黑洞世界模擬器：玩家可控制太空船靠近黑洞，進入事件視界後觸發傳送到未知世界的概念場景。

## 設計原則

- 第一版重點是展示完整且可玩的地球模式，而不是三個模式平均鋪開。
- 玩家不需要猜現在要做什麼。長期任務由右側任務中心引導，即時危機由畫面警告標語提示。
- 地球模式只允許一顆玩家任務衛星，名稱為 `PlayerSat`。所有發射、微調、避碰與任務判定都作用在這一顆衛星。
- 畫面中的框框只顯示在玩家選取的目標上。未選取的衛星、碎片、任務目標或未知物體不自動顯示框。
- 背景採用「遠方太陽系背景」方向：地球仍是核心，背後有遠方太陽光、星場與少量行星剪影，但不能干擾衛星、軌道、警告文字與選取狀態。

## 主選單與模式切換

遊戲開啟後先進入主選單。主選單提供三個模式選項：

1. 地球外太空模擬器
2. 太陽系模擬器
3. 黑洞世界模擬器

玩家可以用滑鼠點選，也可以用 `1`、`2`、`3` 直接進入模式。各模式中按 `Esc` 回到主選單。程式層級以 `GameMode` 狀態管理主選單與三個模式，避免地球衛星任務、太陽系飛船與黑洞飛船狀態互相混在一起。

## 地球外太空模擬器

### 單一 PlayerSat 規則

現有可重複發射多顆使用者衛星的規則改為單一 `PlayerSat`：

- 發射前，玩家調整的是 `PlayerSat` 的預覽軌道。
- 按 `L` 後，場景中只建立或更新這一顆 `PlayerSat`。
- 發射後，所有軌道微調、避碰燃燒、任務完成判定都作用在 `PlayerSat`。
- 若玩家重新發射，系統應更新 `PlayerSat` 的軌道，不再堆疊 `UserSat-1`、`UserSat-2` 之類的多顆玩家衛星。

這個規則能讓任務中心、風險事件與報告輸出都指向同一個清楚的玩家目標。

### 軌道分層任務

軌道分層任務是發射前任務。右側任務中心顯示玩家這次要把 `PlayerSat` 部署到哪一種軌道層：

- LEO 觀測任務：部署低軌道衛星，用於地球觀測。
- MEO 通訊任務：部署中軌道衛星，用於資料轉傳。
- GEO 定點任務：部署高軌道衛星，用於長期監測。

第一版使用遊戲內縮放後的軌道範圍，不追求真實公里比例：

- LEO：85 到 145
- MEO：146 到 230
- GEO：231 到 330

玩家透過 `Tab` 選擇發射參數，用左右鍵調整軌道半徑。當預覽軌道進入任務要求範圍，任務中心提示「軌道符合任務要求，按 L 發射」。發射後系統再驗證一次，若 `PlayerSat` 位於指定軌道層，任務完成。

### 軌道分層視覺

地球模式中加入 LEO / MEO / GEO 三層軌道輔助圈或半透明帶。這些輔助視覺必須比衛星與事件警告更低調，作為玩家調整軌道的參考，而不是畫面主角。

### 即時事件系統

地球模式在發射後進入監測狀態。監測狀態下，一次只允許一個即時事件存在，避免畫面混亂。

第一版即時事件有兩種：

- 太空碎片接近事件
- 未知物體掃描事件

事件出現時，畫面中直接顯示警告標語。右側面板仍保留詳細資訊，但玩家的第一眼提示來自螢幕警告。

### 太空碎片接近事件

當碎片接近 `PlayerSat` 或目前風險高於安全門檻時，畫面顯示紅色或橘色警告：

```text
COLLISION WARNING
Debris-A17 is approaching PlayerSat
Press A to perform avoidance burn
```

玩家按 `A` 觸發避碰。避碰永遠調整 `PlayerSat`，不移動其他衛星或碎片。

避碰成功流程：

1. `PlayerSat` 顯示推進火焰或光暈。
2. 畫面短暫顯示舊軌道與新軌道。
3. `PlayerSat` 在 1.5 到 2 秒內平滑移動到新軌道。
4. 動畫結束後重新計算距離與風險。
5. 系統顯示結果，例如「Avoidance success: distance improved from 28.0 to 91.0」。

若當前沒有可用避碰方案，按 `A` 時顯示明確回饋，例如「No safe avoidance burn is available. Adjust orbit or continue monitoring.」

### 未知物體掃描事件

未知物體會顯示在畫面中。第一版一次只出現一個未知物體，名稱為 `Unknown-01`。它使用與一般衛星、碎片不同的顏色，例如青藍色或紫色。

事件出現時，畫面顯示青藍色警告：

```text
UNKNOWN OBJECT DETECTED
Unknown-01 entered the monitoring zone
Select Unknown-01 and press C to scan
```

掃描不是自動發生。玩家必須選取 `Unknown-01`，再按 `C` 或點擊掃描按鈕。掃描時可以顯示短暫掃描動畫，例如圓形波紋、閃爍或進度條。掃描完成後揭露結果：

- 若為衛星，物體轉為一般衛星並加入顯示。
- 若為碎片，物體轉為碎片並加入風險分析。
- 若為高風險物體，掃描完成後可接續太空碎片接近事件。

未選取未知物體時不顯示框。選取後，若它是可掃描目標，顯示青藍色框。

### 選取框規則

框框只出現在玩家目前選取的目標上：

- 紅色或橘色框：選取目標涉及避碰風險。
- 黃色框：選取目標是任務相關目標。
- 青藍色框：選取目標是未知物體，可以掃描。
- 普通物體若沒有任務、風險或掃描狀態，不顯示框；右側面板可以顯示目前選取名稱。

未選取的物體不因為風險分析、任務或未知狀態自動加框。

### 背景方向

地球外太空模擬器使用 Option A「遠方太陽系背景」：

- 背景有遠方太陽光與星場。
- 可以加入少量遠方行星剪影，暗示地球位於太陽系環境中。
- 地球、衛星、軌道分層、警告標語與選取框仍保持最高可讀性。
- 背景不使用過亮的大面積星雲，不讓玩家誤認為可互動物件。

## 太陽系模擬器

太陽系模式第一版是可操控原型，不只是靜態入口。

玩家進入後看到太陽、主要行星與行星軌道，並可控制太空船在場景中移動。第一版目標是「自由漫遊 + 星體介紹」：

- `W` / `S` 控制前進與後退。
- `A` / `D` 控制左右轉向或平移。
- 太空船靠近行星時，顯示該星體名稱與簡短介紹。
- 太空船皮膚系統先不實作，但架構上保留未來擴充空間。

星體介紹不需要百科全書式長文。每個星體以 2 到 3 行簡短說明為主，讓展示時玩家能快速理解。

## 黑洞世界模擬器

黑洞模式第一版也是可操控原型。

玩家進入後，太空船面向一個黑洞。玩家可以控制太空船靠近或遠離黑洞。當距離小於事件視界門檻時，觸發傳送流程：

1. 畫面顯示「Entering the event horizon...」。
2. 場景短暫扭曲、變暗或出現光暈。
3. 切換到未知世界概念場景。

未知世界第一版只需要是概念預覽，不展開完整玩法。重點是讓玩家感覺黑洞模式有一個清楚的核心互動：靠近黑洞並觸發傳送。

## UI 與提示

地球模式右側面板改為任務中心，固定顯示：

- 目前部署任務
- 目標軌道層
- `PlayerSat` 狀態
- 當前事件細節
- 下一步操作

即時事件另以螢幕上方或中央的警告標語呈現。警告標語要短、明確，直接告訴玩家按什麼鍵或選什麼目標。

底部控制列更新為三模式共通與當前模式專用提示。主選單、太陽系、黑洞、地球模式都應有清楚的返回主選單提示。

## 資料與狀態邊界

建議新增或整理以下狀態：

- `GameMode`：主選單、地球、太陽系、黑洞。
- `EarthMissionState`：地球模式任務進度、目標軌道層、`PlayerSat` 是否已部署。
- `ActiveEvent`：目前即時事件類型、目標物體、事件狀態。
- `AvoidanceAnimationState`：避碰動畫進度、起始軌道、目標軌道。
- `UnknownScanState`：未知物體是否存在、是否被選取、掃描進度、揭露結果。
- `ShipState`：太陽系與黑洞模式的太空船位置、方向與速度。

這些狀態應分開管理，避免主迴圈變成所有模式共用的大型條件堆疊。

## 錯誤與邊界情況

- 若玩家在沒有避碰事件時按 `A`，顯示「No active collision warning」。
- 若玩家在沒有選取未知物體時按 `C`，顯示「Select Unknown-01 before scanning」。
- 若玩家在掃描完成後再次按 `C`，顯示「Object already identified」。
- 若玩家未進入目標軌道層就按 `L`，可以允許發射，但任務中心顯示部署未達標；也可以提示玩家先調整。第一版採用提示但不阻止，避免玩家卡住。
- 若玩家按 `Esc` 回主選單，該模式可重置或暫存狀態。第一版採用重置模式狀態，讓展示流程乾淨。

## 測試與驗證

需要保留現有風險分析與任務狀態 smoke tests，並新增以下驗證：

- `PlayerSat` 重複發射不會增加多顆玩家衛星。
- 軌道半徑會正確判定 LEO / MEO / GEO。
- 太空碎片接近事件只在一個即時事件槽中出現。
- 按 `A` 只會改變 `PlayerSat` 的軌道。
- 未知物體只有選取後按 `C` 才會掃描。
- 掃描完成後未知物體會轉成衛星或碎片。
- 主選單能進入三個模式，`Esc` 能回主選單。
- 太陽系與黑洞模式中太空船可移動。

手動驗證需包含：

- 進入地球模式，確認背景是遠方太陽系背景，且警告文字清楚。
- 完成一次指定軌道部署任務。
- 觸發一次避碰事件並觀看避碰動畫。
- 觸發一次未知物體掃描並確認揭露結果。
- 進入太陽系模式操控太空船靠近行星。
- 進入黑洞模式操控太空船靠近黑洞並觸發傳送。

## 範圍外

第一版不包含：

- 太空船皮膚選擇。
- 完整太陽系探索任務線。
- 完整黑洞後世界玩法。
- 多個未知物體同時出現。
- 多個即時警告事件同時存在。
- 真實天文比例模擬。

這些可以在三模式架構穩定後再擴充。
